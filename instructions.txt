
ברוכה הבאה לאפליקציה :) 

1. פתחו חלון שורת פקודה (CMD) בתיקיית הפרויקט

2. צרו והפעילו סביבה וירטואלית:

    python -m venv venv
    venv\Scripts\activate

3. התקינו את הדרישות:

    pip install -r requirements.txt

4. הריצו את האפליקציה:

    python api.py

5. גשו בדפדפן לכתובת:

    http://127.0.0.1:5000
